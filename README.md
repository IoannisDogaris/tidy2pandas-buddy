# tidy2pandas‑buddy
A lightweight web tool that automatically translates **tidyverse R code** into equivalent **pandas Python code**. Built end‑to‑end inside Lovable, tidy2pandas‑buddy helps analysts move seamlessly between ecosystems with clear, readable translations.

---

## 🚀 Overview
**tidy2pandas‑buddy** is designed for data analysts, scientists, and learners who work across both R and Python. The app provides instant, side‑by‑side translation of tidyverse pipelines into idiomatic pandas code — making it easier to learn, compare, or migrate workflows.

---

## ✨ Features

### 🔄 Side‑by‑side translation
Paste tidyverse code on the left and immediately receive pandas code on the right.

### 🧠 Verb‑accurate mappings
Supports common tidyverse verbs, including:
- `filter()`
- `mutate()`
- `select()`
- `group_by()`
- `summarise()`
- `%>%` pipes → pandas method chaining

### 📋 Copy‑to‑clipboard output
One‑click copying makes it easy to reuse the generated Python code.

### 📘 Built‑in example
Includes a ready‑to‑run tidyverse snippet so users can test the translator instantly.

### 🧼 Clean, minimal UI
Focused layout that emphasizes readability and workflow speed.

---

## 🛠 Technical Approach
- Built entirely through **Lovable’s AI‑powered app generation**.
- No manual coding required — UI, logic, and interactions were scaffolded automatically.
- Translator tested using realistic tidyverse pipelines to ensure correct pandas output.
- Lightweight architecture makes the tool easy to host, share, and extend.

---

## 🎯 Purpose
tidy2pandas‑buddy serves as:

- a **bridge** for R users transitioning into Python,
- a **learning tool** for understanding how tidyverse concepts map to pandas,
- a **workflow accelerator** for analysts switching between languages,
- a **portfolio project** demonstrating rapid prototyping, product thinking, and cross‑ecosystem fluency.
